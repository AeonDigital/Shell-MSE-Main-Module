# Synopsis

Identifies when the passed array has the indicated index.



&nbsp;

# Parameters

## int index

- aka       : -i --index
- hint      :  
  Searched index.


&nbsp;

## array arrayName

- aka       : -a --arrayName
- hint      :  
  Name of the array where the search will be done.



&nbsp;

# Returns bool

`1` if found or `0` if not.
