# Synopsis

Identifica quando o valor indicado existe no array alvo.



&nbsp;

# Parameters

## string value

- aka       : -v --value
- hint      :  
  Valor procurado.


&nbsp;

## array arrayName

- aka       : -a --arrayName
- hint      :  
  Nome do array em que a pesquisa deve ser feita.


&nbsp;

## bool caseSensitive

- aka       : -cs --caseSensitive
- default   : 1
- hint      :  
  A pesquisa deve ser feita em `case sensitive`?
