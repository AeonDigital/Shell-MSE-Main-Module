# Synopsis

Printa os valores ou chaves do array ordenados conforme indicação passada.



&nbsp;

# Parameters

## array arrayName

- aka       : -a --arrayName
- hint      :  
  Nome do array alvo.


&nbsp;

## string target

- aka       : -t --target
- default   : v
- options   :
  - v: values
  - k: keys
- hint      :
  indica se deve ordenar as chaves ou os valores dos arrays.


&nbsp;

## string order

- aka       : -o --order
- default   : a
- options   :
  - a: asc ascendant
  - d: desc descendant
- hint      :
  Indica a forma de ordenação a ser feita.



&nbsp;
