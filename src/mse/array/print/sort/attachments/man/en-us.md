# Synopsis

Prints the values or keys of the array sorted as indicated.



&nbsp;

# Parameters

## array arrayName

- aka       : -a --arrayName
- hint      :  
  Name of the target array.


&nbsp;

## string target

- aka       : -t --target
- default   : v
- options   :
  - v: values
  - k: keys
- hint      :
  Indicates whether to sort array keys or values.


&nbsp;

## string order

- aka       : -o --order
- default   : a
- options   :
  - a: asc ascendant
  - d: desc descendant
- hint      :
  Indicates the form of sorting to be done.



&nbsp;

# Returns string

The items will be written one on each line.
