# Synopsis

Indica quando uma string inicia com uma substring.



&nbsp;

# Parameters

## string substring

- aka       : -ss --substring
- hint      :  
  Substring que está sendo procurada.

&nbsp;


## string string

- aka       : -s --string
- hint      :  
  String na qual a busca está sendo feita.



&nbsp;

# Returns bool

`1` se `true` ou `0` se `false`.
